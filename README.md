# finserv-datahub-poc

A proof-of-concept data hub for the financial services domain.

## Tech Stack

- **React 18** + **TypeScript**
- **Vite 6** (dev server & build)
- **Tailwind CSS 3**
- **Recharts** for charts
- **Lucide React** for icons

## Project Structure

```
.
├── app/                # React + Vite application
│   ├── src/            # Source code
│   ├── public/         # Static assets
│   ├── index.html
│   ├── package.json
│   ├── tailwind.config.js
│   ├── tsconfig*.json
│   └── vite.config.ts
└── README.md
```

## Getting Started

### Prerequisites

- Node.js 18+ and npm

### Install

```bash
cd app
npm install
```

### Run (development)

```bash
npm run dev
```

The app starts on the Vite dev server (default `http://localhost:5173`).

### Build (production)

```bash
npm run build
```

### Preview production build

```bash
npm run preview
```

### Lint

```bash
npm run lint
```
